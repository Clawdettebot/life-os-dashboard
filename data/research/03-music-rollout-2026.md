# Music Rollout 2026 - Research Notes

## Current Project Status
- Annual music release strategy
- Multi-project rollout (Handsome Pizza, Scamboy Color)

## Release Strategy Best Practices (2026)

### Pre-Release (4-6 weeks out)
- [ ] Build anticipation with teasers
- [ ] Collector/merch bundle ready
- [ ] Playlist pitches submitted
- [ ] Press release written
- [ ] Social content calendar mapped

### Release Week
- [ ] Day-of social blast
- [ ] Email list notification
- [ ] Discord community activation
- [ ] TikTok/Reels clips ready

### Post-Release
- [ ] Content pipeline (vlogs, behind scenes)
- [ ] Remix versions planned
- [ ] Playlist follow-ups
- [ ] Tour/festival feelers

## Bundling Ideas
- Vinyl + digital + sticker pack
- Cassette revival - cheap to make, collectors love them
- Zine included with purchase
- Art prints

## Platform Strategy
| Platform | Content Type | Posting Frequency |
|----------|-------------|-------------------|
| TikTok | 30-60s clips | Daily |
| Instagram | Reels + Stories | Daily |
| YouTube | Vlogs + Lyric videos | Weekly |
| Discord | Community | Daily |

## Metrics to Track
- Stream count by platform
- Playlist additions
- Social engagement rate
- Conversion: stream -> merch
